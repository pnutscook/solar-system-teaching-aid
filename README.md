# 太阳系小学课堂网页教具

这是一个独立静态网页教具，用于老师课堂投屏演示太阳系。它不需要后端，也不依赖远程 CDN，构建后可放到学校内网、本地服务器或 U 盘配套服务器中使用。

## 使用

```bash
npm run lint
npm run build
npm run preview
```

打开终端显示的本地地址即可。

## 内容

- 太阳、八大行星、月球、轨道线。
- 五个讲解步骤：认识太阳系、行星顺序、地球自转、四季变化、月相。
- 两种比例：课堂比例、大小对比。
- 可切换自动运动和真实数值提示。
- 可播放当前步骤的中文讲解音频，使用浏览器内置语音合成。
- 左侧步骤栏、右侧讲解栏和顶部工具栏都可收缩，方便全屏投屏。
